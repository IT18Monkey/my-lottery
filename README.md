# my-lottery

> 使用electron-vue、element-plus实现的彩票应用
> 支持功能
1. 大乐透机选
2. 双色球机选

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
